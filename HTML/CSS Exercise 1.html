Do you know what CSS is? CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

  • Improves Appearance

what are the new features of CSS3?

CSS3 is divided into several separate documents called "modules". Each module adds new capability or extends features defined in CSS2.

  1. Text Effects
  2. 2D/3D Transformations
  3. Animations
  4. Multiple Column Layout
  5. User Interface
  6. Box Model
  7. Backgrounds and Borders
  8. Flexbox
  9. Grid Layout
  10. Filters
